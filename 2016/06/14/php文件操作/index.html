php的文件相关操作

  • php

作为前端，与后台的数据交互必不可少，所以了解一些后台的内容还是很有必要的。本篇主要介绍php中常见的文件处理操作。

打开与关闭文件

1、fopen()

resource fopen (string filename,string mode)

fopen()函数将filename对应的resource绑定到一个stream上。 

$file = fopen("1.txt","r+");
echo "$file";

文件的打开模式一共有以下几种，如图：

推荐根据实际情况，使用r、w、a三种模式。

2、fclose()

bool fclose (resource handle)

将handle指向的文件关闭。如果成功则返回true，失败则返回false。

虽然每个请求最后都会自动关闭文件，但明确地关闭打开的所有文件是一个好的习惯。 

fclose($file);

读写文件

1、filesize()

filesize函数作用是获取指定文件的大小，参数是一个文件名（含路径）。返回值是文件大小，单位是字节。 

$filesize = filesize("1.txt");
echo "$filesize";

2、fgetc()

fgetc函数是读取文件的一个字符，读完之后，文件光标会自动往后移动一位。 

for($i=0;$i<$filesize;$i++){
    $str = fgetc($file);
    echo "$str";
}

3、fgets()

fgets函数一次读取一行（以\n换行符作为判断的标准） 

$str = fgets($file,7);
echo "$str";

其中第二个参数也可以指定长度。

4、fread()

fread函数，读取指定长度的内容。读够了length或者到文件结尾会自动结束。

$str = fread($file,10);
echo "$str";

5、fseek()

fseek是用于更改文件光标位置的，有三个参数，文件，偏移量，参考位置。参考位置有三中，SEEK-SET,SEEK-CUR,SEEK-END。

fseek($file,9);
$str = fgetc($file);
echo "$str";

6、file_get_contents()

file_get_contents读取文件内容（全部），其实是fileopen,fgets(包含循环),fclose的封装。 

$allContents = file_get_contents("1.txt");
echo "$allContents";

7、file()

file读取文件中的内容，生成一个数组（以\n作为分隔）

$array = file("1.txt");
print_r($array);

8、file_put_contents()

file_put_contents函数是往指定文件里写入内容，默认打开方式是w,会清空原有内容。file_put_contents是fopen,fwrite,fclose的封装。

$str = "ahahaha";
file_put_contents("1.txt",$str);

9、fwrite()

fwrite作用是往文件里写入指定数据。如果写了参数3，限制了长度，那么我们参数2只会写入指定长度的字符。返回值是一个整数，告知你写入了多少个字符。

fwrite($file,"speak");
