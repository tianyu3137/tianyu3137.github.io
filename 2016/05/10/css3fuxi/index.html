css3的新属性

  • css3

CSS用于控制网页的样式和布局，CSS3是最新的CSS标准，带来了很多新特性。

CSS3边框

CSS3新添加了3个与边框有关的属性，分别是border-radius,box-shadow,border-image（圆角，盒阴影和边框图片）

border-radius是最为常用的属性，当设置为border-radius:50%时，即一个圆。border-radius最多可以设置4条属性值，分别对应左上，右上，右下，左下。（属性值可以为百分比，或固定的px值。） 

div{
    border:2px solid;
    border-radius:25px;
}

box-shadow属性被用来添加阴影 

div{
    box-shadow:10px 10px 5px #333
}

一般只添加4个值，分别是h-shadow(水平阴影的位置，允许负值。)，v-shadow（垂直阴影的位置，允许负值。），blur（模糊距离），color（阴影颜色）。

border-image属性允许你制定一个图片作为边框！（基本不用！） 

div{
    border-image:url(border.png) 30 30 round;
}

CSS3渐变（常用）

CSS3渐变可以让你在两个或多个指定的颜色之间显示平稳的过渡。CSS3定义了两种类型的渐变:线性渐变和径向渐变。

线性渐变（Linear Gradient）: 向下/向上/向左/向右/对角方向
为了创建一个线性渐变，你必须至少定义两种颜色结点。颜色结点即你想要呈现平稳过渡的颜色。同时，你也可以设置一个起点和一个方法（或一个角度。）
语法： background:linear-gradient(direction, color-stop1, color-stop2, …);

从左上角到右下角的线性渐变:

#grad{
    background:-webkit-linear-gradient(left top, red, blue);
}

使用透明度（Transparency）
CSS3渐变也支持透明度(transparency),可用于创建减弱变淡的效果。
为了添加透明度，我们使用rgba()函数来定义颜色结点。rbga()函数中的最后一个参数可以是0-1的值，它定义了颜色的透明度:0表示完全透明，1表示完全不透明。

从左到右的线性渐变，带有透明度：

#grad{
    background:-webkit-linear-gradient(left,rgba(255,0,0,0),rgba(255,0,0,1));
}

径向渐变(Radial Gradient) : 由它们的中心定义。为了创建一个径向渐变，你也必须至少定义两种颜色结点。颜色结点即你想要呈现平稳过渡的颜色。同时，你也可以指定渐变的中心、形状（原型或椭圆）、大小。
默认情况下，渐变的中心是center，渐变的形状是ellipse（椭圆），渐变的大小是farthest-corner（表示到最远的角落）。
语法： background: radial-gradient(center, shape size, start-color, …, last-color);

颜色结点不均匀分布的径向渐变

#grad{
    background:-webkit-radial-gradient(red 5%,green 15%,blue 60%);
}
