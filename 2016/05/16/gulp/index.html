gulp-基于流的自动化构建工具

  • gulp
  • node.js
  • 工具

Gulp.js是一个自动化构建工具，开发者可以使用它在项目开发过程中自动执行常见任务。Gulp.js是基于Node.js构建的，利用Node.js流的威力，你可以快速构建项目并减少频繁的IO操作。

1.创建项目文件夹

3.安装gulp插件
查询glup插件地址
在此推荐几个比较常用的，gulp-concat(合并文件)，gulp-jsmin，gulp-htmlmin,gulp-cssmin(压缩html，css，js文件)
安装方法也比较简单，终端输入命令，（以gulp-concat为例，）
npm install gulp-concat –save-dev
输入此命令即可安装gulp-concat插件，其他插件同理，只需改变名字。
