Css的导航布局

  • css3
  • html5
  • 布局

几种常见的导航条。

1.中间自适应，两边固定宽度的导航。（原理是两边利用浮动，设置固定宽度或百分比，根据需求调整，中间不设置宽度，通过margin撑起。） 

.div1{
    width:250px;
    height:200px;
    float:left;
    background-color:blue;
}
.div2{
    width:250px;
    height:200px;
    float:right;
    background-color:purple;
}
.div3{
    height:200px;
    background-color:red;
    margin-left:250px;
    margin-right:300px;
}   

<div class="div1"></div>
<div class="div2"></div>
<div class="div3"></div>

*注意3个div的排序，中间的div放在最后。

效果如下图，

2.中间固定宽，两边自适应。(原理是利用两边div浮动到两边，宽度给50%，通过margin负值给中间的div留出空间。） 

body{
    width:100%;
    overflow:hidden;
}
.div1{
    width: 50%;
    height: 100px;
    float: left;
    margin-left: -120px;
    background-color: blue;
}
.div2{
    width: 50%;
    height: 100px;
    float: right;
    margin-right: -120px;
    background-color: gray;

}
.div3{
    width: 200px;
    height: 100px;
    margin: 0 auto;
    background-color: black;
}  

<div class="div1"></div>
<div class="div2"></div>
<div class="div3"></div>

需要注意的是给body添加一个宽100%,超出隐藏属性，否则浏览器下方会出现滚动条！

  1. 多列分栏导航条

先上效果图，
（中间为透明logo,整个导航具有position-fixed的属性，固定top 0，logo+红色区域宽度固定,黄色区域宽度自适应）

这个导航布局相对繁琐，具有多个颜色分明的区域，代码如下: 

*{
    margin:0;
    padding:0;
}
.div1{
    width: 100%;
    height: 75px;
    position: fixed;
    left: 0;
    top: 0;
}
nav{
    width: 875px;
    height: 75px;
    margin:0 auto;
    position: relative;
}
h2{
    width: 75px;
    height: 75px;
    float: left;
    background-image: url(1.png);
    background-repeat: no-repeat;
}
ul{
    width: 800px;
    list-style: none;
    background-color: red;
    overflow: hidden;
    float: right;
    position: relative;
}
nav::before{
    content: '';
    height: 75px;
    width: 100%;
    position: absolute;
    left: -100%;
    background-color: gold;
}
nav::after{
    content: '';
    height: 75px;
    width: 100%;
    position: absolute;
    right: -100%;
    background-color: gold;
}
ul li{
    float: left;
    width: 25%;
    height: 75px;
}  

<div class="div1">
    <nav>
    <h2></h2>
        <ul>
            <li>1</li>
            <li>2</li>
            <li>3</li>
            <li>4</li>
        </ul>
    </nav>
</div>

此例的难点在于如何处理两边黄色的区域，在代码中使用了伪元素before和after来进行处理，但其实换成div也是可以的。使用了负定位+超出隐藏的办法来进行处理。在Css布局中，灵活应用超出隐藏进行负定位（或者margin）可以解决不少看似繁杂的问题。

总结，html+css是网页布局的基础，但是入门虽易，精通却不易，仍需要不断的探索与思考！
