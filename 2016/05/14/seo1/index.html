前端与SEO

  • SEO
  • html
  • 前端

SEO中文译为搜索引擎优化。通俗理解为，通过总结搜索引擎的排名规律，对网站进行合理优化，使你的网站在百度和Google的排名提高。

前端是构建网站中非常重要的一个环节，负责页面的HTML+CSS+JS，优化好这几个方面会为SEO工作打下一个良好的基础。

前端优化seo，主要有以下几种方法:

合理利用meta标签

meta标签具有两个对seo比较有用的属性，分别是keywords和description. 

<meta name=“keywords” content=“”>关键词以10-15个为宜   

<meta name=“description” content=“120|150 words” />描述部分控制在120-150个字符为宜。

语义化书写HTML代码，符合W3C标准。

对搜索引擎来说，直接面对的是网页HTML代码，如果代码书写得语义化，搜索引擎就会很容易读懂网页要表达的意思。例如合理利用h1-h6标签，列表形式的代码使用ul或ol，重要的文字使用strong等等。

为图片加上alt属性

<img src="dog.jpg" alt="狗" title="lovely" />

当网络速度很慢，或图片地址失效时，可以体现alt属性的作用，让用户在图片没有显示的时候知道图片的作用。需要强调的地方加上title属性。

尽量使用外链CSS和JS

这样做的好处是可以把内容、表现和行为分离开，保证代码整洁的同时方便维护。CSS放在文件头部，JS放在文件尾部，可以使用工具对CSS和JS文件进行压缩。

CSS Sprites

使用CSS Sprites可以把网页用到的切片合成到一张图上，这么做既减少了HTTP请求数，又使得样式图片一次加载，避免网页“白”的尴尬。
