伪类

伪类有4个，分别是link,visited,hover和active,其中最常用的是hover伪类。语法格式为 :link/visited/hover/active.

:active 效果为鼠标点击时，元素增加特效，鼠标松开时，特效消失。

:hover 当我们需要对某一对象添加鼠标悬浮时改变样式，可以用到hover(多说一句，:hover的好搭档cursor属性，当属性值为pointer时，光标覆盖目标时会变成手型。cursor还有url属性，可以为其自定义图片地址)

:focus 当我们需要让点击后的元素一直拥有某些样式，这时用:active就不行了，因为松开鼠标样式会消失。:focus可以用在<input>标签上

:visited 将特殊的样式添加到被访问过的链接

关于这四个伪类的顺序有一个比较幽默的说法，爱恨法则 love&hate,即link(l),visited(v),hover(h),active(a).

伪元素

伪元素共有5个，分别是before,after,first-line,first-letter,selection.其中最常用的是before和after。语法格式为 ::before/::after/::first-line/::first-letter/::selection(虽然:号也可以被浏览器解析，但是为了区分伪类和伪元素，建议使用::)

::before 在某元素之前插入某些内容

::after 在某元素之后插入某些内容

::first-letter 将特殊的样式添加到文本的首字母

::first-line 将特殊的样式添加到文本的首行

::selection 改变选中区域的背景色和文本颜色（只可改变这两个属性）

下面通过几段代码来演示伪元素的用法与效果
first-line与first-letter:

div{
    width: 200px;
    height: 200px;
    background-color: blue;
}
div::first-line{
    font-size: 22px;
    color: red;
}
div::first-letter{
    color: white;
}    

<div>
付款单就疯狂倒计时福克斯<br />
反抗军大煞风景<br />
防空洞里是否<br />
</div>

总结

伪类的效果可以通过添加一个实际的类来达到，而伪元素的效果则需要通过添加一个实际的元素才能达到，这也是为什么他们一个称为伪类，一个称为伪元素的原因。
