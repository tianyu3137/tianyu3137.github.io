6.如何垂直居中一个元素？(1.已知高度100X100；2.未知高度的)

<div class="box box1"><span>我是垂直居中元素</span></div>
非标准答案。
方法1：display:table-cell     // table-cell
.box1{
      text-align:center; 
      display:table-cell;
      vertical-align:middle;
}

方法2：display:flex         //flexbox方法
.box2{
      display:flex;
      justify-content:center;
      align-items:center; 
      text-align:center;
}

方法3: translate(-50%,-50%)       // 需要定位
.box3{ position:relative;}
.box3 span{
      position:absolute; 
      left:50%; 
      top:50%; 
      -webkit-transform:translate(-50%,-50%); 
      width:100%; 
      text-align:center;
}

方法4: display:inline-block     // 需要有参照物
.box4{
    text-align:center;
    font-size:0;
}
.box4 span{
      vertical-align:middle; 
      display:inline-block; 
      font-size:16px;
}
.box4:after{
      content:'';
      width:0;
      height:100%;
      display:inline-block;
      vertical-align:middle;
}

方法5: margin:auto            //  flexbox的方法
.box5{
    display:flex;
    text-align:center;
}
.box5 span{
    margin:auto;
}

方法6: display:-webkit-box     //  flexbox的方法
.box6{
    display:-webkit-box;
    -webkit-box-pack:center;
    -webkit-box-align:center;
    text-align:center;
}

7.var foo=0,bar=1;foo=foo||bar;这段代码运行后的foo的值是什么？ 

a&&b:将a,b转换成Boolean类型，再执行逻辑与  

a||b:将a,b转换为Boolean类型，再执行逻辑或  

此题答案为1.  

当||时，找到为true的分项就停止处理，并返回该分项的值，否则执行完，并返回最后分项的值。  

当&&时，找到为false的分项就停止处理，并返回该分项的值。

35.块级元素和行内元素的区别以及如何清除浮动 

块元素自占一行，支持宽高属性设置，支持margin,padding值。
行元素在一排显示，不支持设置宽高，内容撑开宽高，不支持纵向的padding值。

36.CSS中position属性有哪几个值，区别是什么？ 

有fixed,relative,absolute,static.
absolute,生成绝对定位元素，相对于static定位以外的第一个父元素进行定位。
fixed，生成绝对定位元素，相对于浏览器窗口进行定位。
relative，生成相对定位的元素，相对于其正常位置进行定位。
static,没有定位。元素出现在正常的流中(忽略top,bottom,left,right或者z-index声明)

37.CSS选择器的种类及优先级 

常用的有，id选择器，class选择器，标签选择器和后代选择器。
优先级为:id选择器>class选择器>标签选择器
后代选择器优先级需根据其前面的元素选择器来判断。
