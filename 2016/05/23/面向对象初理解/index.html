JavaScript的面向对象浅解

  • JavaScript
  • 面向对象

JavaScript是一种基于对象(object-based)的语言，你遇到的所有东西几乎都是对象。此篇简述面向对象的基础知识点。

数据类型

JS中分别有两种数据类型，基本类型和引用类型。基本类型指的是简单的数据段，而引用类型指那些可能由多个值构成的对象。

基本数据类型共有6个，分别是字符串(String)、数值(Number)、布尔(Boolean)、未定义(Undefined)、空(Null)和对象(Object).

基本数据类型使用typeof来进行检测，例： 

var zz = 123;
console.log(typeof zz)   // Number  
var vv = "abc";
console.log(typeof vv)   // String

注意，如果使用typeof来判断引用类型数据，那么永远都返回Object(Chrome浏览器除外，Chrome浏览器现已支持typeof值为function。但数组仍返回Object)。例： 

var a = [1,2,3]
console.log(typeof a)   //Object
var b = function(){
    console.log('dfs')
}
console.log(typeof b);   // Object(Chrome下为function)

引用数据类型只有两个，分别是函数(Function)与数组(Array)。

引用数据类型使用instanceof来进行检测，返回的是Boolean值，只有true和false两个结果。例：

var c = new Person();
alert(c instanceof Object)   // true
alert(c instanceof Person)   // true
alert(c instanceof Dog)   // false

注意：如果使用instanceof来检查基本类型，则永远返回false。

var d = '333';
alert(d instanceof Number)   // false

函数

关于函数，首先我们要了解的是，函数的三种创建方式，分别是普通函数、匿名函数和构建函数。

1.普通函数。

function aaa(){}
aaa()

上面的例子即是一个普通函数，通过给予函数一个名称来进行调用。
（需要注意的是，普通函数属于一级公民，优先级最高。即浏览器在读取JS文件的时候，会先预览一遍，把文件中的普通函数提到文件的最上方，之后再依照从上到下的顺序进行解析。）

在常见的面试题中，常用普通函数的特性进行挖坑！

2.匿名函数 

var fn1 = function(){}   
fn1()

上面即一个匿名函数，与普通函数不同的是，函数本身是没有名称的，而是通过定义一个变量进行赋值，使其等于那个函数。
